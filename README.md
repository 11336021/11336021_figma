# LPTDIVING Website

A responsive diving website showcasing diving articles, spots, and gear information.

## Features
- Responsive design for all devices
- Article showcase with pagination
- Diving spots gallery
- Contact forms
- About us section

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- Responsive Design

## Live Demo
Visit: https://11336021.github.io/11336021_figma_final/

## Structure
- `index.html` - Homepage
- `articles-1.html` - Articles page 1
- `articles-2.html` - Articles page 2  
- `join.html` - Contact/Join page
- `css/style.css` - Main stylesheet
- `images/` - Image assets
