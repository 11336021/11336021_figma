# LPTDIVING Website

一個專業的潛水網站，展示潛水文章、景點和裝備資訊。

## ✨ 功能特色
- 響應式設計，支援各種裝置
- 文章展示與分頁功能
- 潛水景點畫廊
- 聯絡表單
- 關於我們專區
